# data-structures-practice
# Data Structures Practice

C programs for all lab sessions from the Data Structures course (2025–26).

Each lab folder contains a `.c` file written from scratch as exam preparation.
Programs are compiled and tested using GCC.

---

## Labs

| Lab | Topic | Status |
|-----|-------|--------|
| Lab 01 | Array Operations (Insert, Delete, Search) | ✅ |
| Lab 02 | Recursive Binary Search | ✅ |
| Lab 03 | Bubble Sort & Insertion Sort | ✅ |
| Lab 04 | Hybrid Sort (Quick + Insertion) | ✅ |
| Lab 05 | Hashing with Linear Probing | ✅ |
| Lab 06 | Singly Linked List | ✅ |
| Lab 07 | Circular Singly Linked List | ✅ |
| Lab 08 | Doubly Linked List | ✅ |
| Lab 09 | Stack — Infix to Postfix & Evaluation | ✅ |
| Lab 10 | Binary Tree — Traversals, Sum, Leaf Count | ✅ |

---

## How to Run

```bash
gcc filename.c -o output
./output
```

Requires GCC. On Windows, use MinGW or WSL.

---

## Topics Covered

- Arrays — insertion, deletion, linear search
- Searching — recursive binary search
- Sorting — bubble, insertion, hybrid (quick + insertion)
- Hashing — division method, linear probing, load factor, rehashing
- Linked Lists — singly, circular singly, doubly
- Stack applications — infix to postfix conversion, postfix evaluation
- Trees — level-order construction, inorder/preorder/postorder, sum, leaf count
